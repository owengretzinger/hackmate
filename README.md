## About The Project

HackMate is an AI-powered hackathon companion designed to help participants excel in their projects. Browse winning hackathon projects to get inspiration, generate documentation using AI that has the context of your entire codebase, and draft your pitch.

### Built With

- **Next.js** with T3 Stack (create-t3-app): A powerful framework for building modern web applications with server-side rendering and static site generation.
- **TypeScript:** Ensures type safety and code maintainability.
- **Tailwind CSS:** A utility-first CSS framework for rapid and consistent styling.
- **tRPC:** Provides a type-safe way to communicate between the frontend and backend.
- **Vertex AI:**  Google's managed machine learning platform for building and deploying AI models.

## Getting Started

### Prerequisites

- Node.js (v20 or higher)
- npm
- Google Cloud Platform account with a project

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/owengretzinger/hackmate.git
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Configure environment variables
   ```sh
   cp .env.example .env
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
